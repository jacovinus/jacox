Onboarding

Connecting to Ollama

Jacox connects natively to your local Ollama instance. Ensure it's running and the API is accessible.

# Update config.yaml
llm:
  provider: "ollama"
  model: "mistral"
  ollama:
    base_url: "http://localhost:11434"
Model Setup

Running Mistral Locally

Mistral is a powerful, lightweight model perfect for local-first apps. Pull it with one command.

# In your terminal
ollama pull mistral
ollama run mistral

API Basics

Jacox is OpenAI-compatible. Use your favorite tools or curl to chat with your local models.

curl http://localhost:8080/v1/chat/completions \
  -H "Authorization: Bearer sk-dev" \
  -d '{"model": "mistral", ...}'
